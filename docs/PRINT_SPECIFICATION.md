# Print Specification

## Card Size
Standard poker/trading card size: 2.5 x 3.5 inches.

## Supported Print Modes

- Standard duplex
- Fold-over
- Three-panel accordion, six faces total

## Print Rules

- Print at 100% / Actual Size
- Do not scale to fit
- Include crop marks
- Include fold marks
- Include safe zones
- Include source/attribution page in exports

## Accordion Format
A complex monster may use three connected card panels. When folded, it appears to be one standard card. When unfolded, it exposes six faces of information.
