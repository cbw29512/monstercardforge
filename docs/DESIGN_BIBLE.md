# Design Bible

## Primary Rule
Data changes. Templates stay consistent.

## Front Cover
The front cover is sacred:

- Full artwork area
- Monster name
- CR badge
- Creature type border
- No combat clutter

## Back / Interior Panels
Combat information is layered by frequency of use:

1. Combat Summary
2. Actions
3. Traits
4. Reactions
5. Legendary Actions
6. Spellcasting
7. Lair / Regional Effects
8. Notes / QR / Source

## Readability Rule
No hidden combat information. No microscopic text. No page flipping.

If content does not fit, the engine selects a larger layout.

## Card Formats

- Standard card: front + back
- Fold-over card: front/back connected for cutting and folding
- Accordion card: three cards attached, six faces total, folding into one 2.5 x 3.5 card

## Color System
Border color is by creature type. Rarity/importance can be a separate gem/pip later.
