# Print Orientation Decision

## Problem

A printable card must support two real home-printing workflows.

## Mode A: Cut-and-Fold

This is the default mode.

- Card front and card back are side by side.
- A spine sits between them.
- The user cuts one connected rectangle.
- The user folds on the spine.
- The folded piece becomes one thick front/back card.
- The user laminates after folding.

This works on any regular printer.

## Mode B: Duplex Front/Back

This is optional for printers with reliable double-sided printing.

- Fronts print on one page.
- Backs print on the reverse side.
- Alignment must be tested.
- The user cuts single cards after printing both sides.

## Product Rule

Every standard card must offer both modes, but the site should recommend cut-and-fold first because it is more reliable for home users.

## Boss Monsters

Boss monsters need a separate decision path because they may not fit on one front/back card. Current options:

- Boss Folio accordion.
- Boss Decklet made from multiple cut-and-fold cards.

Boss Decklet may become the preferred format if laminated accordion folios become awkward.
