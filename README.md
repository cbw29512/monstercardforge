# Monster Card Forge

Monster Card Forge is a print-first and web-ready monster reference system for tabletop RPG Game Masters.

## MVP Focus

The MVP focuses on monster cards only:

- Consistent reusable templates
- Complete combat information
- Standard single cards for simple monsters
- Accordion cards for complex monsters
- SRD 2014 and SRD 2024 separation
- Homebrew monster builder with helper text and examples
- Print-ready preview modes
- Legal attribution and licensing notices

## Run Locally

Open `index.html` in a browser.

No build step is required for this MVP prototype.

## Product Rule

Monster data changes. Templates stay consistent.

If information does not fit, the layout expands. The card does not shrink into unreadable text and combat information is not removed.
